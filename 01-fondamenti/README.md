## Panoramica

Questa sezione copre i mattoni fondamentali di JavaScript. Si osserva che JavaScript è un linguaggio versatile a tipizzazione dinamica che implementa lo standard ECMAScript. Un programma è composto da istruzioni, che a loro volta contengono espressioni. Il JavaScript engine compila ed esegue il codice "al volo" tramite compilazione JIT.

## Percorso di Apprendimento Consigliato

1. Iniziare da **Che cos'è Javascript** per capire la natura del linguaggio
2. Comprendere **ECMAScript** e il suo ruolo come standard
3. Studiare **Programma** per capire cos'è il codice
4. Approfondire **Statement** per comprendere le istruzioni
5. Conoscere **Expression** per capire come si producono valori
6. Esplorare l'**Esecuzione** per comprendere come funziona il JavaScript engine
