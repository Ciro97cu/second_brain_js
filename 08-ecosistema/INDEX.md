# 08 - Ecosistema JavaScript

Questa cartella contiene note su argomenti che riguardano l'**ecosistema JavaScript moderno**, gli strumenti di sviluppo, la compatibilità browser e la distinzione tra JavaScript puro e le API fornite dall'ambiente di esecuzione.

## 📚 Contenuti

### Compatibilità e Build Tools

- **[polyfilling.md](polyfilling.md)** - Replicare funzionalità moderne in browser vecchi
- **[transpiling.md](transpiling.md)** - Convertire sintassi moderna in sintassi compatibile
- **[non-javascript.md](non-javascript.md)** - Distinguere JavaScript puro da API del browser/Node.js

## 🎯 Obiettivi di Apprendimento

Dopo aver studiato questi contenuti, dovresti essere in grado di:

- Capire come rendere il codice moderno compatibile con browser datati
- Distinguere tra funzionalità del linguaggio e API dell'ambiente
- Configurare un ambiente di sviluppo moderno con transpiler
- Implementare polyfill quando necessario
- Riconoscere quando stai usando JavaScript puro vs API del browser
